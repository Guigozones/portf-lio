# Pasta de Imagens dos Projetos

Coloque as imagens dos seus projetos aqui.

## Nomenclatura sugerida:
- `project-1.jpg` - Imagem do primeiro projeto
- `project-2.jpg` - Imagem do segundo projeto
- etc.

## Tamanho recomendado:
- Largura: 400px ou mais
- Altura: 250px ou mais
- Formato: JPG, PNG ou WebP
